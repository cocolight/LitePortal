# LitePortal

### **[更新计划](./docs/开发计划.md)** | **[Docker部署](./docs/Docker部署.md)** | [开发文档](./docs/开发文档.md)

LitePortal是一个简洁高效的网页导航工具，支持内网和外网链接无缝自动切换。可以部署在自己的家庭局域网上。若想二次开发请查看开发文档。

## 一、功能特点

- 简洁直观的导航界面
- 内外网链接智能切换
- SQLite 数据库存储配置
- 多用户支持
- 响应式设计与暗黑模式
- 支持Docker部署

## 二、快速使用

```bash
cd backend
npm install        # 安装依赖
npm start          # 启动服务器
```

访问 http://localhost:8080
