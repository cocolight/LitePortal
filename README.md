# LitePortal

## 一、项目简介

LitePortal是一个简洁高效的网页导航工具，支持内网和外网链接无缝自动切换。可以部署在自己的家庭局域网上，支持私有部署。

🛖项目地址：[LitePortal: nas门户导航](https://gitee.com/yumos/LitePortal)

## 二、功能特点

- 简洁直观的导航界面
- 内外网链接智能切换
- SQLite 数据库存储配置
- 响应式设计与暗黑模式
- 支持Docker部署
- 跨平台打包（Linux、macOS、Windows）
- 生成独立的可执行文件
